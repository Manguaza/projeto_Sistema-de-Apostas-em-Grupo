# Sistema de Apostas em Grupo

Projeto acadêmico de Programação Orientada a Objetos.

O sistema simula uma plataforma onde usuários de um grupo criam apostas entre si usando saldo fictício. Cada participante aceita os requisitos da aposta e deposita um valor simulado. O dinheiro fica bloqueado até que o resultado seja definido. Ao finalizar a aposta, o sistema transfere automaticamente o valor total para o vencedor.

> Observação: o sistema é apenas uma simulação acadêmica. Não utiliza dinheiro real.
